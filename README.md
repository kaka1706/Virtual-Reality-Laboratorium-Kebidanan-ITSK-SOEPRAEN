# Virtual-Reality-Laboratorium-Kebidanan-ITSK-SOEPRAEN

**Virtual Reality Midwifery Laboratory** is a **Virtual Reality (VR)**-based application designed to support immersive midwifery practice learning through 3D simulation. This application was developed as part of a technology-based academic project, primarily to enhance the learning experience of students in the midwifery laboratory.

This project was created by a team of developers from **ITS K Soepraoen**, with the aim of providing interactive VR-based learning media that is more realistic and applicable than conventional media.

## About the Project

Virtual Reality (VR) is a technology that can present realistic simulated environments that help students understand the work processes in midwifery laboratories through direct interaction in the digital realm. This technology allows for practice without spatial limitations and risks to real patients, making it suitable for improving students' skills and readiness before practicing in the real world.

In the context of health education, especially midwifery, VR simulation can strengthen students' understanding of theory while training their clinical skills in a safer and more efficient manner.

Key Features
Some features commonly found in midwifery laboratory VR projects:

- **Interactive 3D Simulation** — Displays the laboratory space and practice tools in three dimensions.
- **Step-by-Step Instructions** — Provides guidance on midwifery practices in the correct order of operations.
- **VR Object Interaction** — Students can select and use medical objects virtually.
- **Headset/Controller Support** — Compatible with VR devices such as Oculus, HTC Vive, or WebXR devices.
- **Immersive Learning Media** — Provides a realistic learning experience without real risks.

> Note: features depend on the files and implementations in the repo.
